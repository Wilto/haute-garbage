## Getting Started

### 1. Clone this repository:

```
git clone git@github.com:Wilto/wilto-makes-food.git wilto-makes-food
```

### 2. Navigate to the directory

```
cd wilto-makes-food
```

### 3. Install dependencies

```
npm install
```

### 4. Run Eleventy

```
npx eleventy --serve
```

### 5. Run Grunt
```
grunt
```

Or build automatically when CSS changes:
```
grunt watch
```
